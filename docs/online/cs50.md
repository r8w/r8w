Mockup

Save Person Game

- Choose a character which you want to save
	
	- One character is good
	- One character is bad
	- if you save the good character
		- he changes the world
	- if you save the bad character
		- you kill the good character
		- the bad character kills you
	- if you decide not to choose anyone
		- you get rickrolled
		- both character dies
		- you die too
    - if three bad guesses in a row
        - game over
    - if five correct guesses in a row
        - you get reward / game over
